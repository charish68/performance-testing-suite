# E-Commerce Performance Testing Suite

## Objective
To evaluate system performance under moderate and high concurrent user load using Apache JMeter.

## Test Scenarios

### 50 Users Test
- Avg Response Time: 548 ms
- Error Rate: 0%
- Throughput: 20 req/sec

### 200 Users Test
- Avg Response Time: 1873 ms
- Error Rate: 22%
- Throughput: 6 req/sec

## Conclusion
System performs well under moderate load but shows performance degradation and increased failure rate under high concurrency.
